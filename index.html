About Bibscraper

Bibscraper is an open-source project that helps to check out prices of your favorite book from leading online bookstores and choose the cheapest one. Download the zip/tar version and follow the steps.

$ apt-get install scrapy
$ cd <extracted-folder>
$ scrapy deploy scrapysite -p scrapBib
$ scrapy crawl <crawler-name> -a isbn=<isbn_number>

is one of flipkartspider , homeshop18spider or infibeamspider.

Make sure PYTHONPATH environmental variable is set correctly to your extracted folder. Check out this link for setting correct environmental variables while using scrapy.

If you need to use the UI, install php compatibile web server , say apache and point DOCUMENT_ROOT to /ui/ or do :

$ cd <extracted_folder>
$ cp -r ui/* /var/www/*
  • Assuming /var/www is the DOCUMENT_ROOT and run http://localhost/bibscraper.html in browser.

Miscellaneous

All crawlers are defined inside /scrapBib/spiders. Check out scrapy documentation to understand how to write crawlers better. Scraped data are JSON formatted currently. You can define your own pipelines. Make sure to add your pipelines to ITEM_PIPELINES in settings.py. Define pipeline classes in /scrapBib/pipelines.py.

All UI related files are in /ui/. UI is written using the cool Twitter bootstrap framework.

scrapy server is assumed to run on 6800 port (default). If you are changing it, change it correspondingly in the curl request URLs in ui/fetchresults.php file too.
